
# Personal Portfolio Website

A full-stack personal portfolio website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- Responsive personal portfolio website
- About Me section
- Skills section
- Projects showcase
- Contact form with backend integration
- Resume download option
- MongoDB database integration
- Fully deployed online

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## Installation

### Clone the repository

```bash
git clone YOUR_GITHUB_LINK
```

### Install backend dependencies

```bash
cd backend
npm install
```

### Start the server

```bash
node server.js
```
